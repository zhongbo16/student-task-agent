# Student Task Agent

Student Task Agent is a local-first AI-assisted task organizer designed specifically for university students. This app aims to help students organize tasks from various sources like syllabi, course materials, and calendars while integrating features for future enhancements like daily study planning.

## Features (MVP)

- Local-first approach
- Manual task input
- SQLite lightweight database
- Streamlit web app for user interface

To be implemented in later versions:
- Syllabus PDF parsing
- Canvas/Quercus integration
- AI-powered task suggestions using the OpenAI API

---

## Run Instructions

1. Clone the repository.
2. Install dependencies from `requirements.txt`.
3. Run `app.py` using Streamlit: `streamlit run app.py`
