# Agent Memory Plan

The system should preserve raw execution data first:

- tasks
- study_sessions
- daily_reviews

Later, the AI Boss can summarize patterns into persistent memory:

- recurring blockers
- task avoidance patterns
- time estimation errors
- preferred management style
- high-friction courses or task types

Raw data should not be overwritten by AI summaries.
AI-generated memory should be editable by the user.
